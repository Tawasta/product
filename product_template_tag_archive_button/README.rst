===================================
Product Template Tag Archive Button
===================================

Add archive-button to Product Template Tag and disable deleting
product.template.tag-records. Use the added Archive-filter to see archived
records.

Configuration
=============
No special configuration needed.

Usage
=====
Just install this module.
